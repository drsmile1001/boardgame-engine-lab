### PlayerApi 註冊匿名玩家

File: packages/backend/src/apis/PlayerApi.ts (line 33)

- 移除 `setRequester` 改成

```ts
//...
await PlayerRepo.set(newPlayer);
const session: Session = {
  id: crypto.randomUUID(),
  playerId: newPlayer.id,
  createdAt: Date.now(),
  expiresAt: Date.now() + SESSION_LIFETIME * 1000,
};
await SessionRepo.set(session);
sessionCookie.httpOnly = true;
sessionCookie.value = session.id;
sessionCookie.maxAge = SESSION_LIFETIME;
sessionCookie.expires = new Date(Date.now() + SESSION_LIFETIME * 1000);
```

- `SESSION_LIFETIME` 放到 `src/Constants.ts` 中，使用 30 天
- 測試 `POST /api/register/anonymous` 的流程，包含 session cookie 的設定
